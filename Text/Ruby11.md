#11.ハッシュ
------------------------------
###11-1.ハッシュとは
ハッシュとは、文字列やシンボルなどをインデックスとして扱うもので、通常数地しか撮れなかった配列のインデックスが文字列を取れるようになる。

ハッシュの生成の仕方は

```Ruby
	p = Hash.new
	p[キー値] = 値
```

とすれば良い。

###11-2.ハッシュの作り方と使用法
```Ruby
	p = {"a" => "b", "c" => "d"} #p["a"] => "b"
	p.keys #=>["a", "c"]
	
```
